  1. conceito de monetização da dívida

    1. evitar que se use emissão de base monetária para financiar gasto público

  2. dívida subscrita pelo sistema financeiro obriga o BC a emitir as mesmas reservas que teria emitido para financia diretamente o tesouro. (André Lara)

    1. subscrita pelo sistema financeiro significa que os bancos com conta no BC, os primary dealers, precisam estar sempre disponíveis para comprar títulos emitidos pelo tesouro

    2. quando esses bancos compram títulos, suas reservas junto ao BC diminuem, teoricamente cria pressão para o juros subir (confirmar isso?)

    3. então, o BC, faz operação de curto prazo com esses bancos no open market onde o BC compra títulos dos bancos em troca de aumentar de volta as reservas. (confirmar isso)

  3. definição de operações compromissadas pelo Lara: emissão de reservas, BASE monetária, para o sistema bancário

    1. ele não fala, mas há uma troca, os bancos entregam ativos e recebem reservas

  4. Lara posiciona que no brasil não existe o sistema de reserva remuneradas

    1. já ouvi falar que as reservas remuneradas servem para arbitrar a disponibilidade que os primary dealers tem para emprestar no REPO market

  5. Lara explica o QE: implementado pelos BC, nada mais é que a expansão de base monetária para que o BC possa socorrer o sistema financeiro

    1. ele pare que nunca viu a definição correta de QE: compra de ativos pelo FED, que aumentar as RESERVAS dos bancos.

    2. de uma forma acaba aumentando a BASE monetária.

  6. ponto central do Lara, quando o sistema financeiro é usado como intermediário para comprar títulos do tesouro, eles decidem a taxa (confirmar isso porque os PD deveriam comprar prontamente a taxa oferecida determinada pelo BC?)

    1. ele fala que os bancos irão lucrar com essas taxas e irão usar as reservas do BC para financiar esse lucro
