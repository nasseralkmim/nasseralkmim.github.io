  1. NÃO usar tags para tópicos

    1. tag dieta pode ser usada para vários assuntos

    2. mas usar ela somente para dietas e não para problemas metabólicos, senssibilidade a insulina e etc

    3. focar em conexões mais precisas e insightful

    4. tags criam grupos que se intersectam

  2. ponto positivo de TAGs

    1. uma anotação pode pertencer a vários tópicos

    2. pode ser uma camada adicional de relação que não atrapalha conexão mais a fundo com outras anotações

  3. problema com categorias

    1. exemplo de uso: categoria ou tag "metabolic flexibility" iria coletar todas as anotações com esse tópico coletar todas as anotações com esse tópico.

    2. uma categoria rígida significa que a anotação só fica lá, por exemplo, numa caixa de papel.

    3. mas na era digital, as buscar podem ser dinâmicas, ou seja, o filtro de categoria pode ser desligado ou ligado.

    4. além disso, uma nota pode estar em várias categorias simultaneamente, algo impossível no mundo físico.

    5. categorias criam grupos exclusivos
