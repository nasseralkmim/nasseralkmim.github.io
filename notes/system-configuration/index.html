Swap CAPS LOCK to CTRL

GitHub - ililim/dual-key-remap: Remap any key to any other two keys on Window…

Configure to use as escape (ESC) and control (CTRL) when with other. Perfect for emasc.

remap_key=CAPSLOCK
when_alone=ESCAPE
with_other=CTRL

Tpmiddle

GitHub - Jyny/TPmiddle: Scroll & middle-click with the TrackPoint Configure to use middle button on trackpad. I use version 0.6, it seems that the newer ones are not compatible with thinkpad T440s…

Synaptics driver version 19.0.17.115 is the one that works fine.

Emacs
