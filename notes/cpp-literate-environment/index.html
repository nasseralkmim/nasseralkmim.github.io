Tangle block into file

C++ org-babel block have no support for session. Which means I can not use one block definitions/declarations in another one, like in a Jupyter notebook or python org-babel blocks.

My idea is to save it to a file and import in another block. Using the :tangle filename header argument. Extracting Source Code (The Org Manual).

It seems that :tangle does not automatically creates a folder. To begin, lets start with current directory.

#include <string>
class Widget {
public:
std::string value {"from the class!!"};
};
std::string foo () {return "from the function";}

I don't know why the block execution gives an error ((.text+0x24): undefined reference to `main'), but it works as intended. The file is created after I save the buffer.
