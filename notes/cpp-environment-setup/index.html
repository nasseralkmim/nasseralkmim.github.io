Emacs

Lsp-mode clangd server

clangd is language server that provides intelligent features to emacs (client) via lsp-mode (client-server application). Features include, errors warning, completion, references, navigation and formatting. An alternative to clangd is ccls.

C++ (clangd) - LSP Mode - LSP support for Emacs

sudo apt-get install clangd-9

and

sudo update-alternatives --install /usr/bin/clangd clangd /usr/bin/clangd-9 100

C++ compiler

sudo apt install clang

Org-mode babel

Requirements

  1. compiler available

  2. add C to org-babel-load-languages with

(org-babel-do-load-languages
 'org-babel-load-languages '((C . t)))

Example 

int a=1;
int b=1;
printf("%d\n", a+b);
