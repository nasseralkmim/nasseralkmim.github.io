Changing dataframe inside a function

If you passa a DF to a function, manipulate it and return it. The result is the DF modified. If you want to keep the old DF you need to make a copy of it inside the function.

import pandas as pd
dfold = pd.DataFrame(data=[])
print(dfold)

def manipulate(df):
    df['column'] = 10
    return df

dfnew = manipulate(dfold)
print(dfnew)
print(dfold)                    # old also changed

dfold = pd.DataFrame(data=[])
print(dfold)

def manipulate(df):
    df_n = df.copy()
    df_n['column'] = 10
    return df_n

dfnew = manipulate(dfold)
print(dfnew)
print(dfold)                    # don't change old
